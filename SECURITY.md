# Security Policy

## Supported Versions

This project is no longer under active development. Security updates may not be regularly maintained. Users are advised to review and adapt the code before using it in production environments.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

* Avoid creating public issues for security-related concerns
* Contact the maintainer directly

Please include:

* Description of the vulnerability
* Steps to reproduce
* Potential impact
* Suggested fix (if available)

Contact:

* Maintainer: Ankan
* Email: [your-email@example.com](mailto:ankanchatterjee4855@gmail.com)

## Known Limitations

As this is an academic project, the following limitations may exist:

* Limited input validation in certain areas
* No advanced rate limiting or attack protection
* Dependencies may not always be up to date

---

## Recommendations for Production Use

If you plan to use this project in a real-world environment:

* Add rate limiting and request validation
* Secure environment variables and API keys
* Regularly update dependencies
* Conduct a full security audit

---

## Disclaimer

This project was developed for educational purposes. It is not guaranteed to be secure for production use without additional enhancements.
